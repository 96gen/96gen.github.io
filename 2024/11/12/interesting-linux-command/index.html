五個沒用但是有趣的Linux的指令

今天要介紹幾個 Linux 指令，雖然它們沒有實際用途，但卻充滿趣味！

提醒：以下指令都需要安裝相應的套件，套件名稱和小標題相同。

asciiquarium

輸入 asciiquarium，畫面上會出現由文字組成的魚群在「水中」悠游，營造出有趣的水族箱效果。按 q 即可結束。

cmatrix

想要偽裝成駭客嗎？cmatrix 也許能滿足你的願望。輸入 cmatrix，綠色字元就會像駭客風格的螢幕那樣不斷滾動，按 q 結束。

你也可以切換顏色，例如輸入 cmatrix -C red 切換到紅色，或用 cmatrix -r 讓字元色彩繽紛。

hollywood

覺得 cmatrix 不夠駭客？那麼 hollywood 這個指令更能滿足你的駭客情境需求。輸入 hollywood，螢幕將模擬出各種看似高科技的終端畫面，似乎正在破解防火牆。按下 Ctrl + C 即可結束。

cowsay

撰寫教學文件時想加點趣味？可以試試 cowsay 讓小牛說話！例如，輸入 cowsay "Hello, World!"，會出現以下內容：

除了牛，cowsay 還提供其他角色，使用 cowsay -l 查看完整角色列表。

切換角色時，輸入 cowsay -f 角色名，如要用兔子角色，可以輸入 cowsay -f bunny "Hello, World!"：

bastet

bastet 可以讓你在終端機中玩俄羅斯方塊！輸入 bastet 開始遊戲，按 Ctrl + C 結束。

這些指令可能在工作上幫不上忙，但偶爾用來放鬆心情還是很不錯的。試試看吧！
