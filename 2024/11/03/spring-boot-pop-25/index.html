Spring Boot project-oriented programming（25）安裝Node.js和Visual Studio Code

在先前的專案設計中，我們採用了極簡主義風格，外觀相對簡約。接下來，我們將介紹如何利用流行的前端框架，來設計更加美觀的前端專案。

後端工程師為何要學習前端？

許多後端工程師認為專注於後端開發即可，無需了解前端技術。然而，現實中，台灣的後端工程師往往最終都會轉向全端開發，要求具備前後端的技能。大部分後端工程師的第一份工作，通常是協助公司承接外包項目，這些項目可能僅需後端開發，但也可能包含全端需求。

當分配到全端項目時，工程師需獨立完成前後端的開發工作，因此，掌握前端框架將對職業發展大有裨益。

常用的前端框架

目前流行的前端框架主要是 React 和 Vue.js。這兩個框架在 GitHub 上的星數皆超過二十萬，遠超過 Python 和 Linux 的星數，顯示了它們的受歡迎程度。

安裝 Node.js

不論是 React 還是 Vue.js，都需要使用 Node.js。以下是安裝步驟：

  1. 前往官網 https://nodejs.org/ 。
  2. 點擊「Download Node.js (LTS)」以下載 Node.js，LTS 版本穩定且會持續更新修補漏洞。

安裝完成後，打開命令提示字元，輸入 node -v 確認安裝是否成功。若安裝成功，會顯示類似 v20.16.0 的版本號。

安裝 Visual Studio Code

接著安裝 Visual Studio Code（VS Code），這是一款免費的編輯器，廣受前端工程師使用。前往官網 https://code.visualstudio.com/Download 下載並安裝 VS Code。

以下是幾個推薦的 VS Code 擴充功能，能有效提升開發效率：

  • Chinese (Traditional) Language Pack for Visual Studio Code
    讓 VS Code 界面顯示為中文，方便使用。

  • ES7+ React/Redux/React-Native snippets
    預設新增 JavaScript 或 TypeScript 檔案時，檔案內容是空白的，需要自行補上代碼。此擴充能簡化常見的代碼結構，例如輸入 rafce 會自動產生以下內容：
