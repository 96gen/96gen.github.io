Spring Boot project-oriented programming（20） 從零開始打造電商購物車系統：商品管理的基本功能

在購物車系統中，商品管理是不可或缺的一部分。本教學將指導您如何建立 Product 實體、ProductRepository、ProductService 和 ProductController，並完成新增、刪除及查詢商品的 API 功能。

建立 Product 實體類別

首先，建立 Product.java 來代表商品的基本資料。

建立 ProductRepository

接下來，創建 ProductRepository 來定義與資料庫的基本操作。

建立 ProductService

ProductService 處理商品的業務邏輯，包括新增、刪除及依 ID 查詢商品。

建立 ProductController

最後，在 ProductController 中定義新增、刪除、查詢商品的 API。

測試功能

以下為測試商品管理功能的步驟：

  1. 新增商品

    • 使用 POST 請求至 http://localhost:8080/api/product/ ，在請求的 body 中新增商品資料。

  2. 查詢商品

    • 使用 GET 請求至 http://localhost:8080/api/product/{id} 查詢指定 ID 的商品。

  3. 刪除商品

    • 使用 DELETE 請求至 http://localhost:8080/api/product/{id} 刪除指定 ID 的商品。

商品篩選與分頁功能

進一步擴充 Controller，實現商品篩選（類別、價格範圍）和分頁功能。

Service 層的篩選與分頁實現

在 ProductService 中加入篩選邏輯。

Repository 層的自訂查詢

在 ProductRepository 中定義自訂 SQL 查詢以篩選商品。

一次新增多筆商品

將 ProductController 更新為支持批次新增商品功能。

測試篩選與分頁

  • 分頁查詢: http://localhost:8080/api/product/?pageNumber=0&pageSize=5
  • 按價格篩選: http://localhost:8080/api/product/?minPrice=100&maxPrice=200
  • 篩選類別: http://localhost:8080/api/product/?category=Bags
  • 價格排序: http://localhost:8080/api/product/?sort=price_low

至此，我們已完成商品的基本 CRUD 操作以及篩選、分頁功能！
