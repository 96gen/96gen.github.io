Spring Boot project-oriented programming（10）使用MongoDB打造全端專案 前端與測試

由於專案採用了 Thymeleaf 作為模板引擎，因此所有前端頁面內容應放置於 src/main/resources/templates 目錄下。

以下逐步介紹不同頁面設計與 Thymeleaf 功能的使用方式，並輔以測試操作說明。

1. index.html：顯示客戶列表

說明

  • 使用 <tr th:each> 迴圈將後端傳來的客戶清單逐一顯示於頁面中。
  • 使用 ${} 來綁定資料模型，例如 ${customer.id} 用於顯示客戶 ID。
  • 使用 th:href 動態生成連結。

範例程式碼

2. new_customer.html：新增客戶資料

說明

  • 使用 <form> 元素透過 th:action 將表單資料送至後端。
  • 使用 th:field 綁定資料模型屬性，如 *{firstName}。
  • 使用 th:if 判斷欄位錯誤，並顯示相應錯誤訊息。

範例程式碼

3. edit_customer.html：修改客戶資料

說明

  • 透過 th:action 將修改後的資料送至 /edit/{id}。
  • 同樣使用 th:field 與 th:errors 進行欄位綁定與錯誤提示。

範例程式碼

4. view_customer.html：檢視客戶資料

說明

  • 使用 th:text 顯示客戶的詳細資料。

範例程式碼

測試操作流程

  1. 進入首頁：前往 http://localhost:8080/ 查看現有客戶資料（若無客戶則為空列表）。
  2. 新增客戶：
    • 點擊 **”Add New Customer”**。
    • 嘗試送出空白表單，觀察錯誤提示。
    • 填寫錯誤格式的 Email，再次送出檢查提示。
    • 正確填寫資料後提交，應能看到新客戶出現在首頁列表中。
  3. 檢視單一客戶：點擊 “View” 以查看詳細資訊。
  4. 修改客戶資料：點擊 “Edit” 修改資料並提交，確認首頁資料已更新。
  5. 刪除客戶：點擊 **”Delete”**，確認資料已成功刪除。
