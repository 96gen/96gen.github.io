Spring Boot project-oriented programming（2）Hello World

專案挑戰的第一步是「Hello World」，這是一個建立Spring Boot專案並回應「Hello World」的入門專案。看似簡單，卻能讓初學者逐步熟悉專案環境設置與開發流程。

步驟一：安裝與設置 IDE

首先，我們會使用 IntelliJ IDEA Community ，這是一個免費的 IDE 工具來進行專案開發。在下載和安裝的過程中，請選擇「Community」版，而非「Ultimate」版，後者僅提供30天試用。

下載並安裝完成後，打開 IntelliJ，準備開始新專案。

步驟二：使用 Spring Initializr 建立專案

接著，我們會使用 Spring Initializr ，這是一個官方提供的快速設定工具。Spring Initializr 能幫助你輕鬆生成一個基本的 Spring Boot 專案結構，以下是設定過程：

  1. 選擇 Spring Boot 版本：可使用最新的穩定版，預設通常是最佳選擇。
  2. 填寫專案資訊：設定 Group、Artifact 及 Description，可以自訂專案名稱或「Hello World」。
  3. 選擇 Dependency：只需選擇 Spring Web，因為這是 Web 應用程式的基礎。

點擊 GENERATE 後下載並解壓縮的專案。

步驟三：打開專案並設置 Controller

在 IntelliJ 中打開剛解壓縮的專案，等到右下角的進度條消失，表示專案已經完全載入完成。接下來，在 src/main/java 資料夾下創建一個新 Package，命名為 com.helloworld.controller，並在該 Package 內創建一個名為 HelloWorldController.java 的 Java Class。

HelloWorldController.java 範例

此 Controller 會接收來自 /hello 的請求，並回應「Hello, World!」。

步驟四：執行專案並驗證結果

返回 HelloWorldApplication.java，點擊 public static void main 函數旁邊的綠色三角形啟動專案。首次啟動時，可能需要設置 JDK，選擇版本 17 並安裝 Amazon Corretto。

執行後，打開瀏覽器並進入 http://localhost:8080/hello，你將看到「Hello, World!」的訊息。
