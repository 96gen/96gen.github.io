用Hexo打造靜態Blog

這個網站採用Hexo框架，接下來簡述我是如何完成的。

Hexo

  1. bun install hexo-cli -g #安裝Hexo
  2. hexo init blog #建立Hexo專案
  3. cd blog
  4. bun install #安裝使用的套件

Hexo theme

我不喜歡預設的theme，前往https://hexo.io/themes/查看其它theme。
最終選擇hexo-theme-arknights

  1. git clone https://github.com/Yue-plus/hexo-theme-arknights.git themes/arknights #下載theme
  2. bun install hexo-server hexo-browsersync hexo-renderer-pug hexo-wordcount –save
  3. 將themes/arknights/_config.yml改名成_config.arknights.yml
  4. 修改_config.arknights.yml，呈現變化的部分，其餘維持原樣
  5. 修改_config.yml，啟用arknights theme
  6. hexo server #啟動

我們的靜態Blog就完成了

部署到Github Page

  1. bun install hexo-deployer-git –save
  2. 修改_config.yml
  3. 在Github建立Repository，名稱為96gen.github.io
  4. 進入Repository設定，側邊欄尋找Pages，Source選擇Deploy from a branch，Branch選擇master後儲存
  5. 回到Hexo專案，hexo deploy，上傳到Github

編寫貼文

  1. hexo new post post_name #建立新的post
  2. 前往source/_posts，尋找post_name.md，使用markdown格式開始寫作
    有多個tag時，需要像這樣tags: [Hexo, Blog, Github Page]
