Spring Boot project-oriented programming（1）前言

前言

市面上已有許多關於 Spring Boot 的教學，大多以控制反轉（Inversion of Control，簡稱 IoC）和 Bean 為起點，逐步引導讀者了解 Spring Boot 的架構設計。這些教學對於想要紮實掌握基礎的學習者非常有幫助。然而，從實作一個專案的角度來學習，能夠更有效地應用這些知識。

本篇教學適合以下幾類讀者：

  • 已經具備 Spring Boot 基礎知識，或認為不需要深入基礎講解的學習者。

  • 有 Java 程式經驗，因為我們將用 Java 完成 Spring Boot 專案。

  • 想要獨立完成 Spring Boot 專案的人。

接下來的教學不會詳述 IoC、Bean 或 AOP 的基本概念，而是直接著手於專案開發。

個人經驗分享

在我初次接觸 Spring Boot 時，首先學習的是 IoC 的理論和實作方式，並隨之接觸到 Bean 和導向程式設計（Aspect-Oriented Programming, AOP）。這些概念幫助我完成了第一個 “Hello World” 專案。然而，在後續的專案中，我發現自己很少直接使用這些基礎概念，因為 Spring 框架已經自動化處理了很多事情。這讓我開始思考，若當初能參與一個以實際專案為導向的課程，學習曲線或許會更順暢。

教學內容

為了幫助讀者高效學習，本教學將聚焦於如何完成具體的專案，而非深入討論 IoC、Bean 或 AOP 的原理。課程內容將帶領大家逐步實作專案，透過實戰操作來學習如何搭建 Spring Boot 應用程式。

我們會提供完整的專案範例並解釋每個步驟的用意，讓讀者在實作過程中不僅理解專案運作，也能迅速掌握 Spring Boot 的開發流程。

預計完成的專案

以下是我們將完成的專案名稱及將學習的技巧：

  1. Hello World 專案

  • 建立 Spring Boot 專案

  • 使用 Controller 在網頁上顯示內容

  1. 待辦事項清單

  • 連接 Spring Boot 專案至資料庫

  • 完成 RESTful API

  • 使用測試工具測試 API

  • 打包專案成 jar 檔

  1. 書籍管理系統

  • 在 MariaDB 中建立資料庫

  • 使用 YAML 配置專案設定

  1. 客戶管理系統

  • 在 Spring Boot 中整合 MongoDB

  • 使用 Thymeleaf 建立簡易前端介面

  1. 會員註冊與登入系統
  • 使用 Spring Security 完成權限管理
  1. 購物車系統
  • 整合 Json Web Token（JWT）與 Spring Security 進行權限驗證
  1. AI 專案
  • 在 Spring Boot 專案中整合 Ollama，用於 AI 聊天及影像辨識

此外，為了幫助學習前後端整合，我們還將開發兩個前端專案：

  1. React Restful API 前端
  • 使用 Bun、Tailwind CSS、Axios 及 React-router-dom 完成 React 前端專案
  1. Vue.js 購物車前端
  • 使用 Vue Router、Pinia、並打包 Vue 專案
  • 部署至 Linux 伺服器

結語

透過本教學，讀者將能夠從零開始實作完整的 Spring Boot 專案，掌握從後端 API 到前端整合的全端開發技能。希望這些實作經驗能幫助大家快速提升開發能力，並應用於實際工作中。
