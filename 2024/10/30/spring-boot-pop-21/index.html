Spring Boot project-oriented programming（21） 從零開始打造電商購物車系統：打造購物車

Entity

Cart (購物車)

首先，建立 Cart 實體類別，該類別包含購物車的基本屬性，例如ID、所屬用戶、購物清單、總價格及購買數量等。

註解解釋

  • @OneToOne: 表示Cart和User之間是一對一的關聯關係。

  • @JoinColumn(name = "user_id", nullable = false): 在Cart表格中新增user_id欄位，且不允許空值。

  • @OneToMany(mappedBy = "cart", cascade = CascadeType.ALL, orphanRemoval = true): 表示Cart和CartItem是一對多的關聯關係。

    • cascade = CascadeType.ALL: 對Cart的所有操作都會傳播至其所包含的CartItem。
    • orphanRemoval = true: 當Cart中移除某個CartItem，該項商品在資料庫中也會被刪除。

CartItem (購物車商品項)

CartItem 類別用於表示購物車中的每一項商品，包含ID、所屬購物車、商品、價格、數量等屬性。

註解解釋

  • @ManyToOne: 多對一的關係。
  • @JsonIgnore: 在 JSON 輸出中隱藏此屬性，避免循環引用。
