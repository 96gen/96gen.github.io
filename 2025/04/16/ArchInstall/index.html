ArchLinux初體驗，使用ArchInstall

Arch Linux 安裝教學

我們首先從 ISO 開機，選擇第一個 Arch Linux install medium。
會看到一堆 OK 畫面，這代表系統正在 Boot。等到畫面切換，出現 root@archiso，我們就能開始操作了。

網路測試

確認網路連線正常。

啟動安裝

接下來就輕鬆多了。

基本設定

  1. Mirrors and repositories
    選擇 Select regions，挑選適合的伺服器位置。選完後選 Back。

  2. Disk configuration

    • Partitioning → Use a best-effort default partition layout
    • Filesystem 選擇 ext4
      分割區設定就會自動完成，然後 Back。

  3. 確認 Swap 是否啟用。當 RAM 用滿時，系統會使用 Swap，避免記憶體不足造成當機。

帳號與密碼

  • Root password：設定 root 密碼。
  • User account：輸入使用者名稱、密碼並確認，然後問你要不要 superuser 權限，選 yes，然後 Confirm and exit。

系統與驅動

  • Profile

    • Type 選擇 Desktop
    • 推薦輕量的桌面環境：xfce4

  • Graphics

    • 選擇顯示卡驅動程式
      如果是 Nvidia，建議用 Nvidia (proprietary)，問題比較少。

  • Audio

    • 如果要聲音建議用 PipeWire
    • 不需要聲音可以選 No audio server

  • Network configuration

    • 選 Copy ISO network 開頭的選項

  • Timezone

    • 選擇你的時區，選單很長，慢慢找沒關係。

開始安裝

完成以上設定後，按下 Install，然後選 Yes，系統就會自動安裝。

安裝完成

安裝完成後，會問要不要 chroot，我們安裝時設定過了，選 No。

重新開機，選擇 Arch Linux，然後登入剛才建立的使用者帳號與密碼，就能進入桌面啦！
