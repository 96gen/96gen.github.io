使用yay安裝AUR的軟體

在傳統的套件管理系統像 apt 和 yum 中，官方提供的套件版本通常比較舊。

舉例來說，即使 PHP 已經有了 8.x 穩定版，apt 軟體庫中最新的版本可能還是停留在 7.x。

遇到這種情況，我們通常只能手動下載原始碼自行編譯。

而在 Arch Linux 中，除了官方的 pacman 軟體庫外，還有一個強大的社群套件庫 AUR（Arch User Repository）。

透過 AUR，我們可以直接安裝社群用戶打包好的套件，不用等 pacman 更新，就能使用新版軟體。

以下是安裝 AUR 輔助工具 yay 的步驟：

  1. 安裝 git 和 base-devel
    git 用來下載 yay 的原始碼，base-devel 則是編譯時 makepkg 所需的基本開發套件。

  2. 下載 yay 原始碼

  3. 進入剛下載的 yay 資料夾

  4. 編譯並安裝 yay
